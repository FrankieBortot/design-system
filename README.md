## Install

```bash
yarn add @mia-platform/design-system
```

## Usage

```jsx
import React from 'react'
import {Launcher} from '@mia-platform/design-system'

const Example = () => {
  return (
    <Launcher />
  )
}
```

## Local Development

For local development, you can run:

```bash
yarn install # to install all dependencies
yarn dev
```
